# kubectl-plugins
Handy [plugins](https://kubernetes.io/docs/tasks/extend-kubectl/kubectl-plugins) for kubectl

# install
$ ./install.sh

# plugins

ctx

```bash
# list current-, and available contexts
$ kubectl ctx
# set current context
$ kubectl ctx <name>
```

labels

```bash
# show object labels in table format
$ kubectl labels <type/name>
```
